# 152 Gründe warum ich dich liebe
It displays 152 reasons in a grid. Clicking a reason opens it in a fullscreen view.

## Features
- Light and night mode
- Shuffle button
- Calendar with important dates
- Background particle animation
- Spotify playlist embed

## Open Source
This project is open source and free to use, modify and share.

## Valentine’s Question
There is a temporary Valentine’s Day question inside the code.  
It is designed to appear only once per device using local storage.

## Usage
Simply open the HTML file in any modern web browser.  
An internet connection is required for external libraries and the Spotify embed.
